# VIZORIA AI

**Don't Memorize. Visualize.**

VIZORIA AI transforms boring theoretical concepts into unforgettable animated storytelling experiences using AI.

## 📚 Features

- AI Storytelling Engine
- Visual Learning
- Gamified Education
- Multilingual Narration
- Accessibility Support
- Offline Learning Packs

## 🏗️ Project Structure

```
vizoria-ai/
│
├── frontend/
│   ├── index.html        ← VIZORIA AI HTML interface
│   ├── style.css         ← Frontend styles
│   ├── script.js         ← Client-side JavaScript
│   └── assets/           ← Images, icons, videos

```

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Overall**: TypeScript (where applicable)

## 🚀 Getting Started

### Prerequisites
- Node.js and npm installed

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hariniqo/vizoria-ai.git
cd vizoria-ai
```

2. Install backend dependencies:
```bash
cd backend
npm install
cd ..
```

3. Start the development server:
```bash
cd backend
node server.js
```

4. Open the frontend:
```bash
# Open frontend/index.html in your browser
```
